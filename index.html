    Projects

    Project 1

    I created an interactive chatbot interface that allows users to enter queries, leveraging the Google Custom Search API to fetch and display relevant search results in the chat box. The design ensures a user-friendly experience with a visually appealing layout and responsive interaction. Key features include real-time interaction, where users receive instant responses; dynamic content display with formatted search results including titles, snippets, and clickable links; and a loading indicator that shows a spinner animation while fetching data. Flexible input handling allows users to submit queries by clicking the "Send" button or pressing the Enter key, enhancing convenience. Error handling mechanisms manage API errors gracefully, ensuring users are informed if something goes wrong. The interface is styled with modern CSS, providing a clean, minimalist design that looks good on various devices. Overall, this chatbot interface is both functional and enhances user experience through thoughtful design and interaction elements.

    Project 2

    The Voice Recognition Notes tool is a web-based application that leverages the Web Speech API to facilitate the creation of text notes through voice input. Users can initiate voice recognition by clicking the "Start Recognition" button, which activates continuous listening and transcription of spoken words into text. The transcribed text is then displayed as individual notes on the web page. Users can stop the voice recognition at any time by clicking the "Stop Recognition" button, pausing the transcription process. This tool offers an efficient and hands-free method for taking notes, making it ideal for individuals who prefer dictation over traditional typing. The user interface is simple and intuitive, ensuring a seamless experience for users to capture their thoughts and ideas quickly and effectively.
